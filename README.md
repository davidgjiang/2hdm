# 2hdm
BDT and DNN jupyter notebooks for signal vs background classification for different signal mass ranges of the Pseudoscalar Higgs $A$:
* Low masses: $A = 400$ $GeV$
* Medium masses: $A = 500$, $600$ $GeV$
* High masses: $A = 700$ $GeV$
